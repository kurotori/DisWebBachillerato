Ejemplo:

Este es un ejemplo acerca de como CSS puede aplicarse a las etiquetas normales de HTML.
Las etiquetas de estructura de esta página web han sido modificadas en el archivo CSS vinculado a la misma.

Al especificar propiedades a una etiqueta en CSS, todas las instancias de dicha etiqueta en la página son afectadas por éstas.
También es posible, de esta manera, definir etiquetas exclusivas.
